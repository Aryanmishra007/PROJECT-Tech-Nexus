When you enter details but nothing happens:

CAUSE 1: MySQL not running
  → Start MySQL server (search MySQL in Windows services)
  → Or open MySQL Workbench and check connection

CAUSE 2: Database not created
  → Open MySQL Workbench
  → Run: CREATE DATABASE IF NOT EXISTS projectnexai_ai;

CAUSE 3: Flask not running
  → Stop current Flask process (Ctrl+C)
  → Run: python app.py

CAUSE 4: Wrong URL
  → Make sure you're on: http://localhost:5000
  → NOT http://localhost:5001 or other port

CAUSE 5: Browser cache
  → Clear cache: Ctrl+Shift+Del
  → Hard refresh: Ctrl+F5
