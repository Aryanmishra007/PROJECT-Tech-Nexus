Step 1: Ensure MySQL is running
  MySQL Server: 127.0.0.1:3306
  
Step 2: Create database (in MySQL Workbench or terminal)
  CREATE DATABASE IF NOT EXISTS projectnexai_ai;

Step 3: Copy QR code image
  Run: copy-qr.bat
  Creates: payment-qr.png

Step 4: Install Python packages
  pip install flask flask-cors werkzeug PyPDF2 python-docx mysql-connector-python

Step 5: Run application
  python app.py
  
Step 6: Open browser
  http://localhost:5000
