✅ CHANGES MADE:
   • Admin nav section hidden by default (.admin-only class)
   • updateAdminAccess() function shows/hides based on user role
   • navToScreen() validates admin role before showing analytics
   • Sidebar displays user role (Admin/Student)
   • Backend routes protected with @require_admin decorator

✅ HOW IT WORKS:
   1. When user logs in, role is stored in session
   2. updateAdminAccess(role) is called in loadDashboard()
   3. If role === 'admin', admin nav items become visible
   4. If role !== 'admin', admin nav items remain hidden
   5. Clicking admin link as non-admin shows error toast

🚀 TO GET STARTED
════════════════════════════════════════════════════════════════════════════

Step 1: MySQL Setup
   → Ensure MySQL is running on 127.0.0.1:3306
   → Create database:
      CREATE DATABASE IF NOT EXISTS projectnexai_ai;

Step 2: Copy QR Code
   → Run: copy-qr.bat
   → Creates: payment-qr.png

Step 3: Install Dependencies
   → pip install flask flask-cors werkzeug PyPDF2 python-docx mysql-connector-python

Step 4: Run Application
   → python app.py

Step 5: Open Browser
   → http://localhost:5000
